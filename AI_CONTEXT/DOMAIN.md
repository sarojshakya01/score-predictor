# Prediction Rules

## Exact Score
+15 points
+5 if winning team only correct

## Goal Difference
+5 points will be given if predicted goal difference matches the actual goal difference. e.g. predicted: 4-3, actual: 3-2

+3 points will be given if difference of predicted and actual goal difference is 1. e.g. predicted: 1-3, actual: 1-2

+2 points will be given if difference of predicted and actual goal difference is 2. e.g. predicted: 4-1, actual: 3-2

+1 point will be given if difference of predicted and actual goal difference is 3. e.g. predicted: 4-1, actual: 3-3

+0 Incase of penalty

## Game Duration
+5 Points will be given for 90 minutes prediction is matched.
+10 Points will be given for 120 minutes prediction is matched.
+15 Points will be given for penalty shootout prediction is matched.
+0 points will be awarded if game duration does not match.

## Opening Team (if user's prediction is same as game's opening prediction)
+3 points

## Total Yellow Card(s)
No Point for zero prediction.
Minimum of difference in 3 yellow cards would be considered.
Total number of yellow card(s) are taken from Official source (https://www.fifa.com)
+1 If the difference between predicted yellow card(s) and the total yellow card(s) given in a match is 3.
+2 If the difference between predicted yellow card(s) and the total yellow card(s) given is 2.
+3 If the difference between predicted yellow card(s) and the total yellow card(s) given is 1.
+5 if predicted yellow card(s) match the total number of yellow card(s) given.

## Total Red Card(s) [-2 to 10 points]
No Point for zero prediction.
+10 Points will be given only for exact prediction of total Red Card(s). You should predict with more than zero card(s).
+5 Points will be give if the Red card prediction is more than zero and actual is also more than 0 but not actual.
-2 Points will be deducted if user predicts Red card and no red card is given in the actual game.

Note:

Prediction on each game is mandatory. Failing to predict will result to 0 points.
Prediction for each game will be locked before 1 hour of game time.
